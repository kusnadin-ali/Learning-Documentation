# How to land remote software engineer in japan
  1. Linkedin (about, portofolio, repostt kontent kontent, experience lampirin evidence)
  2. github (minimal 1 commit sehari)
  3. cover letter harus diperhatikan
  
## Interview Process at NeoTechPark
  - Work experience from CV
  - Technologies mastered
  - Committed work hours
  
  # How Japanase Tech Teams Work
  - Deadline are not flexible
    - tanyakan terlebih dahulu kerja team atau gak
  - Avoid ambiguity
    - Everything should be clear: requirements, scope, expectations.
  - Make progress & risks visible early
    - Dont wait. Share update and problems as soon as possible.
  - Built on trust & quality
    - Consistency and transparency = long-term trust.
  
  **Ho Ren So**
  1. Ho = Hokoku - Report
  2. Ren = Renraku - Inform
  3. So = Sodan - Consult
